# What is Infrastructure as Code (IaC)?

## Definition
Infrastructure as Code (IaC) is a practice where infrastructure is managed and provisioned using code instead of manual processes.

---

##  Simple Explanation
Instead of creating servers manually in the cloud console, you write code to create and manage them automatically.

---

## 🔄 Traditional vs IaC

###  Traditional Way
- Manual setup
- Time-consuming
- Error-prone
- Hard to scale

###  IaC Way
- Automated setup
- Fast and repeatable
- Less human error
- Easy to scale

---

##  Benefits of IaC
- Automation
- Consistency
- Version control (Git)
- Faster deployments
- Easy rollback

---

##  Popular IaC Tools
- Terraform
- AWS CloudFormation
- Ansible
- Pulumi

---

##  Real Example
Instead of manually creating an EC2 instance:
>> Use Terraform code to create it in seconds

---

## Conclusion
IaC is a fundamental concept in DevOps that enables efficient and reliable infrastructure management.
