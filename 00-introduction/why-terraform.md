# Why Terraform?

##  Problem Without Terraform
Managing infrastructure manually can lead to:
- Human errors
- Inconsistent environments
- Slow deployments
- Difficult scaling

---

##  Why Use Terraform?

###  1. Automation
Terraform automates infrastructure creation and management.

---

###  2. Consistency
Same code → same infrastructure every time

---

###  3. Multi-Cloud Support
Terraform works with multiple cloud providers:
- AWS
- Azure
- GCP

---

### 4. Version Control
You can store Terraform code in Git and track changes.

---

### 5. Reusability (Modules)
Terraform allows reusable code using modules.

---

###  6. Execution Plan
Terraform shows what will change before applying:
```bash
terraform plan
